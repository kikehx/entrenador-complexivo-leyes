🔑 Configurar Google Drive

Para sincronizar con Google Drive necesitas un Client ID de Google Cloud Console. Solo tienes que hacerlo una vez.

1Ve a console.cloud.google.com → crea un proyecto nuevo
2Menú → APIs y servicios → Biblioteca → activa Google Drive API
3Ve a Credenciales → Crear credencial → ID de cliente OAuth 2.0
4Tipo: Aplicación web — En "Orígenes autorizados" agrega la URL donde abres el archivo (o http://localhost si es local)
5Copia el ID de cliente (termina en .apps.googleusercontent.com) y pégalo aquí:
